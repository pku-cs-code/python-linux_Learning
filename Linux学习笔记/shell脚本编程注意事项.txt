bash调试
sh -x 将执行脚本的内容及输出显示到屏幕上，这个事对调试很有用的参数
  程序代码前面会加+，逻辑错误的脚本 -x会更佳
sh -v 在执行脚本时，先将脚本的内容输出到屏幕上然后执行脚本，如果有错误，也会给出错误提示
sh -n 不会执行该脚本，仅查询脚本语法是否有问题，并给出错误提示

set -n 读命令但不执行
set -v 显示所有读取行
set -x 显示所有命令机器参数

set -x 开启调试功能
set +x 关闭调试功能
可以缩小调试的作用域
